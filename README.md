# AI Mobile Lab

AI Mobile Lab è un progetto Next.js con TypeScript e Tailwind CSS pensato per presentare un laboratorio di prototipazione AI mobile-first.

## Sezioni incluse

- Homepage responsive con hero section.
- Sezione "Come funziona" con un percorso in tre step.
- Sezione "Cosa puoi creare" con esempi di prodotti e workflow AI.
- Sezione "Workflow mobile-first" dedicata a iterazione, accessibilità e test su smartphone.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Script disponibili

```bash
npm run dev
npm run lint
npm run build
```

## Avvio locale

Installa le dipendenze e avvia il server di sviluppo:

```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) per visualizzare l'applicazione.
